**Установите Qt и Qt Creator:**
   - Скачайте с официального сайта Qt
   - Выберите компоненты: Qt, Qt Creator, компилятор (MinGW/MSVC)

**Настройте комплект (Kit):**
Выберите Desktop Qt (MinGW или MSVC)

**Соберите и запустите:**
Нажмите Ctrl+R или кнопку "Run"
